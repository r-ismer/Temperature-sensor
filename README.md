# Temperature sensor

This folder contains a hardware, a software part and a mechanical part. 
The hardware part is a KiCad project which contains the schematic and the pcb layout for the temperature and humidity sensor. 
The software part is a make project which contains all the files needed to build the firwmare that will communicate with the temperature and humidity sensor and send these inforamtion to a station (other project). 
The mmechanical part is a FreeCAD project which contains the top and bottom casing for the temperature ad hhumidity sensor.
